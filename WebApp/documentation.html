BobAI Documentation

Welcome to BobAI! Follow this guide to get up and running quickly, and configure your integrations correctly.

Getting Started

BobAI is an intelligent tooling platform that accelerates your workflow. To begin using our platform:

  1. Create an account using the Sign Up button in the top right.
  2. Navigate to your Dashboard to access your projects.
  3. Provide a GitHub Access Token to enable code generation, repository integration, and automated deployments.

Generating a GitHub Access Token

To let BobAI interact with your repositories and workflows, you must provide a Classic GitHub Access Token. Follow these precise steps:

Why do we need this? We use this token purely to push generated code to your repositories and trigger automated workflows at your request.

Step 1: Navigate to Developer Settings

  1. Log in to your GitHub account.
  2. Click your profile photo in the upper-right corner and select Settings.
  3. Scroll all the way down the left sidebar and click Developer settings (or go directly to github.com/settings/tokens).

Step 2: Generate New Classic Token

  1. In the left sidebar, click Personal access tokens, then select Tokens (classic).
  2. Click the Generate new token dropdown button and select Generate new token (classic).
  3. Give your token a descriptive Note (e.g., BobAI Integration).
  4. Set an Expiration date for the token (we recommend 30-90 days for better security).

Step 3: Assign Critical Permissions

You must check the following boxes for BobAI to function correctly:

  • ☑️ repo — This grants full control of private repositories. It is required for us to read/write code.
  • ☑️ workflow — This allows us to update and execute GitHub Actions workflows.

Step 4: Copy and Save

Scroll to the bottom and click Generate token.

Important: Copy the token immediately! It starts with ghp_. Once you navigate away from the page, you will not be able to see it again.

Go back to BobAI, click on your Account in the top right, and paste your token into the Update GitHub Token field.
